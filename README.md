# Property Management

A monorepo for the Property Management application.

## Structure

```
packages/
  ├── app/      # Frontend application
  └── server/   # Backend server
```

## Getting Started

Install dependencies:
```bash
npm install
```

Run development:
```bash
npm run dev
```

Build all packages:
```bash
npm run build
```

## Packages

- **app**: Frontend application
- **server**: Backend server
